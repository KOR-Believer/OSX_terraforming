# Personal Setting of macOS
> Make like windowsOS.

## ShortCut
- Show Desktop: `Alt`+`D`
- Show Launchpad: `Ctrl`+`Esc`
- Move focus to the Dock: `Alt`+`Tab` (substitution of `Win` + `number key`)
- Select next source in Input menu: F15 (remap key `right alt` as `F15`)
